Automated Lakehouse Pipeline

Data Engineering & Machine Learning Architecture

About the Architecture

This project is a scalable, end-to-end data engineering pipeline built on the Databricks Lakehouse Platform. It programmatically ingests raw dataset variants into a Bronze layer, performs data cleansing and schema enforcement using PySpark and Delta Lake in the Silver layer, and outputs aggregated business intelligence assets in the Gold layer.

To enhance the platform's analytical capabilities, I engineered three original scoring metrics and integrated a machine learning layer utilizing Spark MLlib (KMeans clustering, StandardScaler) to segment collector archetypes. These clusters are directly integrated with Databricks AI/BI to serve end-users with natural language, cluster-based product recommendations.

Recognized as a top submission by Databricks for the Data + AI Summit (DAIS) 2026 Community Challenge.

Tech Stack: Databricks Lakehouse Spark MLlib PySpark Delta Lake Databricks AI/BI
